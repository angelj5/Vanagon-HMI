## Software
### Hardware Connection
+ VCC ----> +5V
+ GND ----> GND
+ DIN ----> D8(COPI)
+ CLK ----> D9(SCK)
+ CS  ----> D6
+ DC  ----> D7
+ RST ----> D5
+ BL  ----> D3
